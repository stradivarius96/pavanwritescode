Web Development

The Internet is the wild west of the modern era; everyone stakes out their own little corner and has the ability to make it whatever they want. This flexibility can be very intimidating, but it doesn’t need to be!

Matthew has experience with a broad range of web solutions to address tasks of any scale. He has used Grails to build large scale corporate web applications, business reporting tools, and RESTful web services.

On the front end, Matthew is currently a fan of using AngluarJS and Bootstrap frameworks for scalable web design, but by the time you read this sentence, it’s likely that there is a new, exciting framework released. Rest assured that Matthew is diligent in staying educated and current with emerging web technologies.

Java Development

I don’t always use Object Oriented Programming. But when I do, I prefer Java. Objects and the design principles surrounding them are an important bedrock for software design.

Java is the foundation of many of the other frameworks listed on this website. Matthew is experienced in Java SE and Java EE principles, Javascript (kind of but not actually Java), UML design, swing, android, Grails, Groovy, Spring, Hibernate, GSP, servlets, and basically anything else Java related. These principles translate to any OO language such as C# and C++, but when it comes to web solutions Java offers a strong ecosystem to build an application.

Mobile Development

Smartphones have changed everything about how the Internet works. Applications need to be designed for consumption on devices as large as a TV and as small as an iPhone. Fortunately, frameworks and web design standards have evolved to provide a quality and consistent experience for users no matter what environment they are using.

Matthew has areas of expertise in mobile development, primarily in scalable web design and android application development. Whether the goal is to create a native or web application, Matthew is able to help guide that decision process and deliver a quality product.

Cloud Solutions

Buying hardware is a thing of the past! These days the magical cloud is here to provide solutions to all ranges of web needs. Think of the cloud as being a service to rent exactly as many pieces of hardware as you need to run your business. It’s now much easier than buying a server, finding a place to plug it in and keep it cooled, and worrying about updates and maintenance. This is a pretty awesome service.

Matthew is knowledgeable in PaaS and IaaS solutions including OpenShift, OpenStack, Heroku, Amazon Web Services, Amazon EC2, Google App Engine, and is willing to pick up whatever other service you might want or need.

Database Design

Every application needs some place to store data. This place needs to be organized, scalable, fast, and reliable. Enter the database; backbone and back end of most things data related.

Matthew has worked with most traditional DBMS including Oracle, Netezza, MongoDB, SQL Server, MySQL, Access, DB2, and can handle anything else that allows an ODBC connection. This experience includes Entity Relationship Diagrams (ERD), SQL writing and optimization, database tuning, account maintenance, ETL, and use of databases as back end support for large and small application.

ETL Development

What happens when you have many different sources in different formats that you want to put in one place? Extract, Transform, and Load. ETL solutions provide the ability to extract data from any data source (web service, database, flat file, etc.), perform operations to filter, enhance, restructure, or otherwise transform the data in any way desired, and load that data into any desired place and format.

Matthew is a experienced Informatica Developer and certified Informatica Administrator. He also is a certified Pentaho developer. His experiences range anywhere from large scale Data Warehousing problems to smaller migrations from old to new corporate recruiting systems.
